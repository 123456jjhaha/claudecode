# Metaprompt 核心原理 - 简明版

## 一句话总结

**Metaprompt = 让 Claude 帮你写提示词的"提示词"**

---

## 工作流程图

```
┌─────────────────────────────────────────────────────────┐
│  阶段 1: 生成提示词模板 (一次性)                          │
└─────────────────────────────────────────────────────────┘

你写：               "帮我写客服回复邮件"
                            ↓
插入到：             Metaprompt（包含6个优质示例）
                            ↓
发送给：             Claude Sonnet 4.5
                            ↓
Claude 输出：        一个专业的提示词模板（包含变量）


┌─────────────────────────────────────────────────────────┐
│  阶段 2: 使用模板处理实际任务 (可重复使用)                │
└─────────────────────────────────────────────────────────┘

你提供：             真实数据（客户邮件、公司名等）
                            ↓
替换到：             提示词模板中的变量
                            ↓
发送给：             Claude Haiku 4.5（或其他模型）
                            ↓
Claude 输出：        最终结果（客服回复邮件）
```

## 关键技术：Few-Shot Learning（少样本学习）

### Metaprompt 的秘密武器

Metaprompt 里包含 **6 个精心设计的示例**：

```
示例 1: 客服对话 → 教会 Claude 如何写"对话型"提示词
示例 2: 文本分类 → 教会 Claude 如何写"分类型"提示词
示例 3: 文档问答 → 教会 Claude 如何写"问答型"提示词
示例 4: 数学导师 → 教会 Claude 如何写"教学型"提示词
示例 5: 函数调用 → 教会 Claude 如何写"工具型"提示词
...
```

### 类比：教孩子写作文

**传统方式**：
```
老师："去写一篇关于春天的作文"
学生："？？？"（不知道怎么写）
```

**Metaprompt 方式**：
```
老师："这里有 6 篇优秀作文，你看看：
      - 第1篇：写景物（如何描写春天）
      - 第2篇：写人物（如何写春游）
      - 第3篇：议论文（如何论述春天的意义）
      ...
      现在，参考这些例子，写一篇关于春天的作文"

学生："哦！我明白了！"（看完示例就会写了）
```

## 实际代码演示

### 完整流程

```python
import anthropic

client = anthropic.Anthropic(api_key="your-key")

# ============ 阶段 1: 生成提示词模板 ============
# （只做一次，或者缓存起来）

# Metaprompt 是一个超长字符串，包含6个示例
metaprompt = """
今天你要为一个AI助手写指令...

<示例1>
任务：客服对话
指令：[一个完整的、专业的客服对话提示词]
</示例1>

<示例2>
任务：文本分类
指令：[一个完整的、专业的分类提示词]
</示例2>

... (还有4个示例)

现在，为这个任务写指令：
{{TASK}}
"""

# 用户定义任务
task = "写一封客服回复邮件"

# 插入任务
prompt_for_generation = metaprompt.replace("{{TASK}}", task)

# 发送给 Claude（第一次对话）
response = client.messages.create(
    model="claude-sonnet-4-5",
    messages=[{
        "role": "user",
        "content": prompt_for_generation
    }],
    max_tokens=4096
)

# 提取生成的提示词模板
template = extract_instructions(response.content[0].text)

print("生成的模板：")
print(template)
# 输出：
# """
# 你将为 {$COMPANY_NAME} 撰写客服回复...
# 客户邮件：{$CUSTOMER_EMAIL}
# 要求：1. 礼貌 2. 解决问题 3. ...
# """


# ============ 阶段 2: 使用模板 ============
# （可以用很多次）

# 第一次使用
email_1 = "我的订单延迟了..."
filled_prompt_1 = template.replace("{$CUSTOMER_EMAIL}", email_1)
filled_prompt_1 = filled_prompt_1.replace("{$COMPANY_NAME}", "京东")

result_1 = client.messages.create(
    model="claude-haiku-4-5",  # 用便宜的模型
    messages=[{"role": "user", "content": filled_prompt_1}],
    max_tokens=2048
)

print("客服回复 1：", result_1.content[0].text)


# 第二次使用（同一个模板）
email_2 = "产品质量有问题..."
filled_prompt_2 = template.replace("{$CUSTOMER_EMAIL}", email_2)
filled_prompt_2 = filled_prompt_2.replace("{$COMPANY_NAME}", "京东")

result_2 = client.messages.create(
    model="claude-haiku-4-5",
    messages=[{"role": "user", "content": filled_prompt_2}],
    max_tokens=2048
)

print("客服回复 2：", result_2.content[0].text)
```

## 关键要点总结

### ✅ Metaprompt 是什么

1. **不是魔法**：就是一个很好的提示词，里面有6个优质示例
2. **教师角色**：通过示例教 Claude 如何写提示词
3. **一次生成，多次使用**：生成的模板可以重复使用

### ✅ 工作原理

1. **第一次对话**：Metaprompt + 你的任务 → Claude 生成专业模板
2. **第二次对话**：模板 + 真实数据 → Claude 产生最终结果

### ✅ 核心优势

1. **节省时间**：不用自己从零写提示词
2. **质量保证**：生成的提示词包含最佳实践
3. **可复用**：一次生成，无限使用

### ✅ 核心技术

1. **Few-Shot Learning**：通过6个示例教会 Claude
2. **结构化输出**：强制按 `<Inputs>` + `<Instructions>` 格式
3. **变量机制**：`{$VARIABLE}` 支持动态替换

## 最后的最后

**Q: 所以 Metaprompt 只是一次对话吗？**

**A: 不完全是！准确地说：**

1. **第一次对话**（生成阶段）：Metaprompt 帮你生成一个好的提示词模板
2. **第二次~第N次对话**（使用阶段）：用这个模板处理实际任务

**核心价值**：通过第一次对话，得到一个可以重复使用的、高质量的提示词模板。

**类比**：
- 第一次对话 = 请大厨教你一个菜谱
- 后续对话 = 按照菜谱做菜（可以做无数次）
